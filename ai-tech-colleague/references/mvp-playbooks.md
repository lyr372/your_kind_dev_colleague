# MVP Playbooks

Use this to suggest a small first test, not a full project.

## Small-Test Template

```text
先验证什么：
准备什么材料：
AI 做哪一步：
人/规则做哪一步：
怎么判断值不值得继续：
```

## Common Small Tests

### Document Q&A

- Pick 10-30 approved documents.
- Prepare 20 real questions.
- Ask AI to answer with source.
- Check whether users can find answers faster.

### Report Drafting

- Prepare one fixed report template.
- Use real data from Excel or BI.
- Ask AI to write only the text explanation.
- Let the owner edit and compare time saved.

### Ticket Triage

- Pick 5 common categories.
- Use 50 historical tickets.
- Ask AI to label category and draft reply.
- Compare with human labels.

### Contract Or Form Extraction

- Pick one document type.
- Extract 5 important fields.
- Ask AI to show source text.
- Let a business owner check accuracy.

### Meeting Summary

- Use existing transcripts.
- Ask AI for summary, decisions, owner, deadline.
- Let meeting owner confirm.

### Approval Assistant

- Pick one low-risk approval type.
- Let AI only mark suspicious points.
- Keep final approve/reject with humans.
